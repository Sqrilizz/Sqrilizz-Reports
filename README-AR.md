## 🎯 الميزات الرئيسية

| الميزة | الحالة | الوصف |
|---|---|---|
| 📝 **بلاغات أساسية** | ✅ | يمكن للاعبين الإبلاغ عن الآخرين |
| 🛡️ **مكافحة البريد العشوائي** | ✅ | يمنع إرسال البلاغات المزعجة |
| 🌍 **3 لغات** | ✅ | الإنجليزية، الروسية، العربية |
| 🎨 **تصميم جميل** | ✅ | ألوان حديثة ورموز تعبيرية |
| 📱 **تيليجرام** | ✅ | إشعارات البلاغات |
| 🔗 **ويب هوك ديسكورد** | ✅ | إشعارات عبر ويب هوك |
| 🙈 **بلاغات مجهولة** | ✅ | وضع خصوصية اختياري |

### 💾 قاعدة البيانات والأداء
- **SQLite افتراضيًا**، مع دعم اختياري لـ MySQL.
- **محسنة لأداء عالٍ** وتأثير ضئيل على الخادم.
- **تبديل قاعدة البيانات** عبر `/report-reload`.
- **تخزين مؤقت (Cache)** لعمليات القراءة السريعة.

---

## 📋 الأوامر الأساسية

### للاعبين
- `/report <player> <reason>` - الإبلاغ عن لاعب

### للمسؤولين
- `/reports` - عرض جميع البلاغات
- `/reports check <player>` - التحقق من بلاغات لاعب معين
- `/reports clear <player>` - مسح بلاغات لاعب
- `/report-reload` - إعادة تحميل الإعدادات

---

## 🆘 هل تحتاج إلى مساعدة؟

### مشاكل شائعة

**❓ الأوامر لا تعمل؟**
- تحقق من الصلاحيات: `reports.admin` لأوامر المسؤولين.

**❓ الألوان لا تظهر؟**
- قم بالتحديث إلى Minecraft 1.16+ لدعم ألوان hex.

**❓ اللغة لا تتغير؟**
- استخدم `/report-reload` بعد تغيير الإعدادات.
