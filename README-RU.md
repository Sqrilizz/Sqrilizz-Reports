## 🚀 Быстрый старт

### Установка
1. Скачайте последнюю версию с [Modrinth](https://modrinth.com/plugin/sqrilizz-report)
2. Поместите JAR в папку `plugins/`
3. Перезапустите сервер
4. Настройте `config.yml` (опционально)
5. Готово! Используйте `/reports` для открытия GUI

### Первые шаги
```bash
# Пожаловаться на игрока
/report ИмяИгрока Причина жалобы

# Открыть GUI репортов (админ)
/reports

# Проверить конкретного игрока
/reports check ИмяИгрока
```

## 📋 Команды

### Для игроков
- `/report <игрок> <причина>` - Пожаловаться на игрока

### Для администраторов
- `/reports` - Открыть интерактивное GUI меню
- `/reports list` - Просмотр репортов в чате
- `/reports check <игрок>` - Проверить репорты игрока
- `/reports clear <игрок>` - Очистить репорты игрока
- `/reports clearall` - Очистить все репорты
- `/report-reload` - Перезагрузить конфигурацию

### Настройка
- `/report-language <en|ru|ar>` - Сменить язык
- `/report-telegram <token|chat> <значение>` - Настроить Telegram
- `/report-webhook <set|remove> [url]` - Настроить вебхуки

## 🎮 Возможности GUI

### Главное меню (`/reports`)
- Просмотр всех игроков с репортами (головы)
- Количество репортов на каждого игрока
- ЛКМ - открыть репорты игрока
- ПКМ - очистить все репорты

### Меню репортов игрока
- Список всех репортов на игрока
- Телепорт к игроку
- Открыть действия для каждого репорта
- Кнопка очистки всех репортов

### Меню действий с репортом
- Телепорт к жалобщику
- Телепорт к нарушителю
- Открыть меню наказаний
- Решить репорт (удаляет его)
- Удалить репорт без действий

### Меню наказаний
- **Предупреждение** - Выдать варн
- **Кик** - Кикнуть с сервера
- **Мут 1ч/1д** - Временный мут
- **Бан 1д/7д** - Временный бан
- **Перманентный бан** - Навсегда
- После наказания: выбор закрыть репорт или оставить

## ⚙️ Конфигурация

### Основные настройки
```yaml
language: ru                    # en, ru, ar
anonymous-reports: false        # Скрыть имя жалобщика
cooldown: 60                    # Секунд между жалобами

report-limits:
  per-player: 3                 # Макс жалоб на игрока
  per-hour: 10                  # Макс жалоб в час
```

### Дизайн
```yaml
design:
  use-hex-colors: true          # Современные цвета (1.16+)
  colors:
    primary: "#FF6B6B"
    secondary: "#4ECDC4"
    success: "#45B7D1"
```

### База данных
```yaml
database:
  type: sqlite                  # sqlite или mysql
  # MySQL опционально - требует драйвер в lib/
```
