# 🎮 Быстрый старт - GUI Меню

## Что нового в версии 7.6.1?

Добавлено полнофункциональное интерактивное GUI меню для управления репортами! Теперь вы можете обрабатывать жалобы игроков прямо в удобном графическом интерфейсе.

## Как использовать?

### 1. Открыть меню репортов
```
/reports
```

### 2. Что вы увидите?
- Список всех игроков с активными репортами
- Голова каждого игрока с количеством жалоб
- Навигация по страницам (если репортов много)

### 3. Действия в главном меню
- **ЛКМ по игроку** → Открыть все репорты на этого игрока
- **ПКМ по игроку** → Быстро очистить все репорты

### 4. Просмотр репортов игрока
После клика на игрока вы увидите:
- Все репорты с деталями (кто, когда, причина)
- Кнопку телепортации к игроку
- Кнопку очистки всех репортов

### 5. Работа с конкретным репортом
Кликните на репорт, чтобы:
- 🔮 Телепортироваться к жалобщику
- 👁️ Телепортироваться к нарушителю
- ⚔️ Наказать игрока (открывает меню наказаний)
- ✅ Отметить репорт как решённый (удаляется)
- ❌ Удалить репорт

### 6. Меню наказаний
Выберите подходящее наказание:
- ⚠️ Предупреждение (warn)
- 🚪 Кик
- 📄 Мут 1 час
- 📖 Мут 1 день
- 🪓 Бан 1 день
- ⚔️ Бан 7 дней
- 🔨 Перманентный бан

## Требования

Для работы команд наказания нужен один из плагинов:
- LiteBans
- AdvancedBan
- BanManager
- EssentialsX

## Права доступа

```yaml
reports.admin: true  # Доступ к GUI меню
```
